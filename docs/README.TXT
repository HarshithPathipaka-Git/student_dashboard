# 🎓 Attendify – Attendance Calculator

A simple and intuitive web app to help students track attendance and stay above the required 75%.

## 🚀 Features

- 📊 Calculate attendance percentage instantly
- 🎯 Supports:
  - Overall attendance
  - Subject-wise attendance
- 🧠 Smart guidance with recommended option
- 📉 Predict how many classes you can miss (bunk calculator)
- 🎨 Clean and user-friendly UI

## 🖥️ Tech Stack

- HTML
- CSS
- JavaScript

## 💡 How It Works

1. Select attendance rule (Overall / Subject-wise)
2. Choose input method
3. Enter required details
4. Get:
   - Attendance percentage
   - Safe / Risk status
   - Bunk prediction
